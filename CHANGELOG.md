# Changelog

All notable changes to this project will be documented in this file.

## [1.1] - 2026-01-28

### Added
- **LM Studio Support**: Added native support for connecting to local LM Studio servers.
- **OpenRouter Support**: Restored integration to access top LLMs via OpenRouter.
- **Cancellation Support**: Implemented "Stop Generation" functionality allowing users to halt streaming responses immediately.
- **Pre-loading**: Added support for triggering model loads (e.g. for Ollama/LM Studio) before chat begins.
